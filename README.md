# ASBIRD FPV Learning Adventure

从零开始的飞行之旅。加入我们，征服蓝天！

这是一个关于 FPV（First Person View）穿越机学习的记录与教学项目。我们旨在记录从入门到精通的完整过程，并将这些知识分享给所有对飞行感兴趣的朋友，特别是像我们一样的大朋友和小朋友。

## 项目特点

本项目不仅仅是一个文档，它包含了三个层次的学习路径：

1.  **模拟训练 (Simulator)**: 炸机不可怕，只要是在电脑里。从模拟器开始，掌握基础操控，建立肌肉记忆。
2.  **硬件组装 (Hardware)**: 认识你的战机。学习焊接、组装、调试，亲手打造属于你的穿越机。
3.  **真实飞行 (Real Flight)**: 戴上眼镜，起飞！学习安全检查、场地评估以及进阶的空中技巧。

## 技术栈

本项目基于 [Next.js](https://nextjs.org) 构建。

-   **框架**: Next.js 15 (App Router)
-   **语言**: TypeScript
-   **样式**: Tailwind CSS
-   **图标**: Lucide React
-   **部署**: Vercel

## 快速开始

首先，运行开发服务器：

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

打开 [http://localhost:3000](http://localhost:3000) 即可在浏览器中查看结果。

你可以通过修改 `app/page.tsx` 来开始编辑页面。页面会在你编辑文件时自动更新。
